# White Deal — Static Demo Prototype

Это полностью статический кликабельный прототип под GitHub Pages.

Внутри:
- `index.html`
- `styles.css`
- `app.js`

## Как загрузить в GitHub вебом

1. Открой репозиторий `white-deal`
2. Нажми **Add file → Upload files**
3. Загрузи **эти файлы в корень репозитория**
4. Включи **Settings → Pages → Deploy from branch → main / root**
