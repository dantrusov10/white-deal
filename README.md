# White Deal Registry v3.1

Статический прототип под GitHub Pages.

Состав:
- `index.html`
- `styles.css`
- `app.js`

Особенности:
- строгий деловой UI: баланс между госреестром и современным IT-инструментом
- раздельные роли: партнер / вендор / арбитраж
- разные разделы и разное наполнение по ролям
- левое меню + центральный реестр + правая панель сведений
- сводка как операционная панель
- отдельная аналитика рынка с обезличенными сигналами
- рейтинги вендоров и партнеров
- споры, нарушения, challenge, жизненный цикл, мультивендорные сигналы

Публикация:
1. Загрузить содержимое папки в корень репозитория `white-deal`
2. Включить GitHub Pages на `main` / `/root`
