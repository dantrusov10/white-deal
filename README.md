# White Deal — static prototype v2

Полностью статический кликабельный прототип для GitHub Pages.

## Что изменено в v2
- усилен строгий светлый дизайн
- добавлены tooltips и пояснения
- добавлен рейтинг вендоров и расширен рейтинг партнеров
- добавлены каталоги вендоров и партнеров по категориям
- дашборды разведены отдельно под партнера / вендора / арбитраж / публичный рынок
- добавлены drill-down карточек, уведомлений, дел и сигналов рынка
- добавлены нарушения, challenge, lifecycle, мультивендорность, evidence и activities

## Как загрузить в GitHub web
Загрузи **содержимое этой папки** в корень репозитория `white-deal`.

## GitHub Pages
Settings → Pages → Deploy from branch → main → /root
