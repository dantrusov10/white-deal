# Deal Registry RF — clickable prototype

Кликабельный фронтовый прототип единого независимого реестра регистрации IT-сделок.

## Что внутри

- 4 роли доступа:
  - Партнер / интегратор
  - Вендор
  - Арбитраж
  - Публичный обзор рынка
- Предзаполненные синтетические данные во всех ключевых разделах
- Кликабельные сценарии:
  - регистрация сделки / проекта
  - продление защиты
  - подача жалобы
  - просмотр карточек сделок
  - подтверждение / отказ со стороны вендора
  - работа арбитража
  - публичные рейтинги и обезличенная лента рынка
- Без бэка: ничего не сохраняется по-настоящему, но сценарии склеены так, чтобы выглядеть как рабочая система

## Запуск локально

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Для GitHub Pages

Репозиторий сделан на Vite + React. Для простого размещения на GitHub Pages можно:

1. Загрузить репозиторий на GitHub
2. Подключить GitHub Actions / Pages build
3. Либо локально собрать `npm run build` и выложить содержимое папки `dist`

## Замечания

- Это именно инвесторский / демонстрационный прототип
- Данные синтетические
- Логика переходов псевдо-реальная
- Реального API, авторизации и базы данных нет
