# White Deal v3 — static prototype

Статический кликабельный прототип для GitHub Pages.

## Что внутри
- index.html
- styles.css
- app.js

## Как использовать
1. Загрузить содержимое этой папки в корень репозитория GitHub.
2. Включить GitHub Pages: Settings → Pages → Deploy from branch → main → /root.
3. Открыть сайт.

## Что покрывает v3
- роли: партнер / вендор / арбитраж / рынок
- сводка по роли
- реестр регистраций
- реестр споров
- каталог вендоров
- каталог партнеров
- рейтинги вендоров и партнеров
- аналитика рынка с обезличенными сигналами
- правая панель карточки объекта
- пошаговая форма новой регистрации
