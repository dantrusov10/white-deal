# White Deal Registry v3.3

Статический прототип под GitHub Pages.

Содержимое:
- index.html
- styles.css
- app.js
- README.md

Что добавлено в v3.3:
- индекс достоверности регистрации
- уровень защищенности регистрации
- сигналы пересечения / дублей
- формализованный жизненный цикл регистрации
- протокол арбитражных решений
- влияние решений на рейтинг
- аналитика чистоты рынка, доли споров, плотности регистраций

Как загрузить:
1. Открыть репозиторий `white-deal`
2. Загрузить содержимое папки в корень
3. В GitHub Pages выбрать:
   - Deploy from branch
   - main
   - /root
