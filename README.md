# White Deal Registry v3

Статический кликабельный прототип под GitHub Pages.

## Что внутри
- index.html
- styles.css
- app.js

## Как открыть
Просто загрузить файлы в корень репозитория GitHub и включить Pages:
- Settings → Pages
- Deploy from branch
- main / root

## Что реализовано
- официальный реестровый layout
- сводка
- реестр регистраций
- реестр споров
- каталог участников
- рейтинги вендоров и партнеров
- аналитика рынка
- журнал действий
- правая панель карточек
- процедурные модальные окна
- подсказки
- синтетические данные
